# MI Productor Prject

## Laravel - Backend

-   php version: 8.1
-   Laravel version : 10

## Vue - Frontend

-   Vue version : ^3.2.37
-   node version : v16^

# How to set up the Project

-   Install node modules

`npm run install`

-   Install laravel Packages

`composer install`

-   Create .env file from .env.example

`cp .env.example .env`

-   Create the DB called 'productor'

-   Run the migration

`php artisan migrate`

-   Run the Seed

`php artisan db:seed`

# How to run the project.

-   Run backend

`php artisan serve`

-   Run Frontend

`npm run dev`
